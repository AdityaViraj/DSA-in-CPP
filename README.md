# ⚔️ DSA-in-CPP

My journey of mastering **Data Structures & Algorithms in C++** – for interviews, contests, and systems work.

## 📂 Structure

Each topic has its own folder:

- `arrays/` – basics, two pointers, sliding window  
- `strings/` – hashing, KMP, tries  
- `linked-list/` – classic problems  
- `stacks-queues/` – monotonic stack, queues  
- `trees/` – BST, traversals, LCA  
- `graphs/` – BFS, DFS, Dijkstra, DP on graphs  
- `dp/` – 1D, 2D DP, knapsack, LIS etc.
